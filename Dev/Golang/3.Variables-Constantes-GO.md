# Go – Variables et Constantes

Ce chapitre fait suite directe aux précédents et explique en détail la **déclaration des variables et des constantes en Go**, en s’appuyant sur la documentation **W3Schools (Go Variables & Go Constants)**.

---

## 1. Variables en Go

Une **variable** est un espace mémoire nommé qui permet de stocker une valeur.

En Go, chaque variable possède :
- un **nom**
- un **type**
- une **valeur**

---

## 2. Déclaration d’une variable avec `var`

### Syntaxe générale

```go
var nom type = valeur
```

### Exemple

```go
var age int = 30
var nom string = "Alice"
```

Le type est obligatoire lors de l’utilisation de `var`, sauf si la valeur permet une inférence.

---

## 3. Inférence de type

Go peut **déduire automatiquement le type** à partir de la valeur fournie.

```go
var age = 30        // int
var nom = "Alice"  // string
```

Le compilateur détermine le type à la compilation.

---

## 4. Déclaration courte avec `:=`

Go propose une syntaxe courte, très utilisée, uniquement **à l’intérieur des fonctions**.

```go
age := 30
nom := "Alice"
```

### Règles importantes

- `:=` déclare **et** initialise la variable
- Le type est automatiquement inféré
- Impossible à utiliser hors d’une fonction

---

## 5. Déclarer plusieurs variables

### Sur plusieurs lignes

```go
var a int = 1
var b int = 2
```

### Sur une seule ligne

```go
var a, b int = 1, 2
```

### Avec inférence

```go
var a, b = 1, "Go"
```

---

## 6. Valeurs par défaut (Zero Values)

En Go, une variable non initialisée reçoit une **valeur par défaut** :

| Type     | Valeur par défaut |
|----------|-------------------|
| int      | 0                 |
| float64  | 0.0               |
| string   | "" (chaîne vide)  |
| bool     | false             |

### Exemple

```go
var x int
fmt.Println(x) // affiche 0
```

---

## 7. Réaffectation des variables

Une variable peut changer de valeur **tant que le type reste identique**.

```go
age := 30
age = 31 // valide
```

Invalide :

```go
age = "trente et un"
```

---

## 8. Portée des variables

### Variables locales

Déclarées à l’intérieur d’une fonction.

```go
func main() {
    x := 10
}
```

### Variables globales

Déclarées en dehors des fonctions.

```go
var version = "1.0"
```

Les variables globales doivent être déclarées avec `var`.

---

# Go Constants

Une **constante** est une valeur **immuable** : elle ne peut pas être modifiée après sa déclaration.

---

## 9. Déclaration d’une constante

### Syntaxe

```go
const nom = valeur
```

### Exemple

```go
const PI = 3.14
const APP_NAME = "MonApp"
```

Les constantes sont évaluées à la compilation.

---

## 10. Constantes typées et non typées

### Constante non typée

```go
const x = 10
```

Le type est déterminé lors de l’utilisation.

### Constante typée

```go
const y int = 10
```

---

## 11. Déclaration multiple de constantes

```go
const (
    A = 1
    B = 2
    C = 3
)
```

---

## 12. Différences entre variables et constantes

| Variable          | Constante          |
|-------------------|--------------------|
| Peut changer      | Immuable           |
| `var` ou `:=`     | `const` uniquement |
| Valeur modifiable | Valeur fixe        |

---

## 13. Bonnes pratiques

- Utiliser `:=` pour les variables locales
- Utiliser `const` dès qu’une valeur ne change pas
- Donner des noms explicites
- Utiliser les MAJUSCULES pour les constantes globales
