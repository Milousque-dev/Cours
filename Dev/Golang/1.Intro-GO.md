## 1. Qu’est-ce que Go ? (Go Home)

Go, aussi appelé **Golang**, est un langage de programmation développé par **Google** et publié en **2009**.

### Objectifs de Go

Go a été conçu pour résoudre plusieurs problèmes rencontrés avec d’autres langages :

- Temps de compilation trop longs
- Code complexe et difficile à maintenir
- Mauvaise gestion de la concurrence

Go vise donc à être :

- **Simple** à lire et à écrire
- **Rapide** à compiler et à exécuter
- **Fiable** pour les applications modernes

---

## 2. À quoi sert Go ?

Go est principalement utilisé pour :

- Applications **backend**
- **APIs REST**
- **Microservices**
- Outils systèmes
- Applications cloud (Docker, Kubernetes sont écrits en Go)

Il est très populaire dans les entreprises travaillant à grande échelle.

## 3. Caractéristiques principales de Go (Go Introduction)

### 3.1 Langage compilé

Go est un **langage compilé** :

- Le code source est transformé en un programme exécutable
- Les erreurs sont détectées **avant l'exécution**
- Les performances sont élevées

### 3.2 Typage statique

Chaque variable a un **type fixe** :

```go
var age int = 30
```

Cela permet :
- Moins d'erreurs à l'exécution
- Un code plus sûr

---

### 3.3 Syntaxe simple

Go élimine beaucoup de complexité inutile :

- Pas de classes traditionnelles
- Pas d'héritage complexe
- Peu de mots-clés

Exemple :

```go
fmt.Println("Bonjour Go")
```

---

### 3.4 Gestion automatique de la mémoire

Go intègre un **Garbage Collector** :

- Libération automatique de la mémoire
- Moins de bugs liés à la mémoire

## 4. Installer Go (Go Get Started)

### 4.1 Téléchargement

Télécharge Go depuis le site officiel :

```
https://go.dev
```

Installe la version adaptée à ton système (Windows, macOS, Linux).

---

### 4.2 Vérifier l’installation

Dans un terminal :

```bash
go version
```

Tu devrais voir s'afficher la version de Go installée.

## 5. Ton premier programme Go (Go Get Started)

Voici le programme Go le plus simple :

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello World")
}
```

---

## 6. Explication détaillée du code

### 6.1 `package main`

- Chaque fichier Go appartient à un **package**
- `main` indique que le programme est exécutable

---

### 6.2 `import "fmt"`

- `fmt` est un package standard
- Il permet l'affichage (`Println`, `Printf`, etc.)

---

### 6.3 `func main()`

- Point d'entrée du programme
- Toujours exécutée en premier

---

### 6.4 `fmt.Println()`

- Affiche du texte dans la console
- Ajoute automatiquement un retour à la ligne

---

## 7. Exécuter un programme Go

### Méthode 1 : Exécution directe

```bash
go run main.go
```

### Méthode 2 : Compilation

```bash
go build main.go
./main
```

---

## 8. Structure de base d’un projet Go

Un projet Go simple :

```
mon-projet/
├── go.mod
└── main.go
```

### Initialiser un module

```bash
go mod init mon-projet
```

---

## 9. Règles importantes en Go

- Le code est **sensible à la casse**
- Les accolades `{}` sont obligatoires
- Les instructions ne nécessitent pas de `;`
- Le code doit être **formaté correctement**

Commande de formatage :

```bash
go fmt
```
