# Go – Data Types (Types de données)

Ce chapitre poursuit le cours sans revenir sur les variables ou les constantes. Il se concentre uniquement sur les **types de données de base en Go**:

- Basic Data Types
- Boolean
- Integer
- Float
- String

---

## 1. Les types de données en Go

Go est un langage à **typage statique** : chaque valeur manipulée possède un type bien défini.

Les **types de données de base** en Go sont :

- `bool`
- `int`, `int8`, `int16`, `int32`, `int64`
- `uint`, `uint8`, `uint16`, `uint32`, `uint64`
- `float32`, `float64`
- `string`

Ces types couvrent la majorité des besoins courants.

---

## 2. Type booléen (`bool`)

Le type `bool` représente une valeur logique.

### Valeurs possibles

```go
true
false
```

### Exemple

```go
var isOnline bool = true
isAdmin := false
```

### Utilisation courante

- Conditions (`if`, `for`)
- États logiques
- Résultats de comparaisons

---

## 3. Types entiers (Integer)

Les entiers servent à représenter des **nombres sans décimales**.

---

### 3.1 Types entiers signés (`int`)

| Type  | Taille  |
|-------|---------|
| int8  | 8 bits  |
| int16 | 16 bits |
| int32 | 32 bits |
| int64 | 64 bits |

Le type `int` dépend de l’architecture (32 ou 64 bits).

### Exemple

```go
var age int = 30
var temperature int16 = -5
```

---

### 3.2 Types entiers non signés (`uint`)

Les entiers non signés ne peuvent contenir que des valeurs positives.

| Type   | Taille  |
|--------|---------|
| uint8  | 8 bits  |
| uint16 | 16 bits |
| uint32 | 32 bits |
| uint64 | 64 bits |

### Exemple

```go
var score uint = 100
var pixels uint8 = 255
```

`uint8` est souvent utilisé pour représenter des données binaires.

---

## 4. Types à virgule flottante (Float)

Les types `float` permettent de représenter des **nombres décimaux**.

---

### 4.1 Types disponibles

| Type    | Précision |
|---------|-----------|
| float32 | simple    |
| float64 | double    |

`float64` est le type recommandé par défaut.

---

### 4.2 Exemple

```go
var prix float64 = 19.99
taux := 0.2
```

---

### 4.3 Attention à la précision

Les nombres flottants peuvent entraîner des **erreurs d’arrondi**.

```go
fmt.Println(0.1 + 0.2) // 0.30000000000000004
```

Ce comportement est normal et commun à de nombreux langages.

---

## 5. Type string

Le type `string` représente une **chaîne de caractères**.

---

### 5.1 Déclaration

```go
var nom string = "Alice"
message := "Bonjour Go"
```

---

### 5.2 Chaînes multi-lignes

Go permet les chaînes multi-lignes avec les backticks `` ` `` :

```go
texte := `Bonjour,
Bienvenue en Go
Bonne journée`
```

Le contenu est conservé tel quel (espaces et retours à la ligne inclus).

---

### 5.3 Accéder à un caractère

Une chaîne est une **suite d’octets**.

```go
s := "Go"
fmt.Println(s[0]) // affiche 71 (code ASCII de 'G')
```

En Go, accéder directement à un caractère retourne un `byte`, pas un `string`.

---

## 6. Conversion entre types (Type Casting)

Go n'effectue **aucune conversion implicite** entre types différents.

### Exemple

```go
var a int = 10
var b float64 = float64(a)
```

Invalide sans conversion explicite :

```go
var c float64 = a
```

---

## 7. Choisir le bon type

Bonnes pratiques recommandées par Go :

- Utiliser `int` pour les entiers standards
- Utiliser `float64` pour les nombres décimaux
- Utiliser `bool` uniquement pour des états logiques
- Utiliser `string` pour du texte
