# Opérateurs en Go

Les opérateurs sont utilisés pour effectuer des opérations sur des variables et des valeurs.

## L'opérateur d'addition (+)

L'opérateur `+` additionne deux valeurs :

```go
package main
import ("fmt")

func main() {
    var a = 15 + 25
    fmt.Println(a) // Affiche: 40
}
```

Le même opérateur peut aussi être utilisé pour additionner :

- Une variable et une valeur
- Deux variables entre elles

```go
package main
import ("fmt")

func main() {
    var (
        sum1 = 100 + 50   // 150 (100 + 50)
        sum2 = sum1 + 250 // 400 (150 + 250)
        sum3 = sum2 + sum2 // 800 (400 + 400)
    )
    fmt.Println(sum3) // Affiche: 800
}
```

## Catégories d'opérateurs en Go

Go divise les opérateurs en plusieurs groupes :

1. Opérateurs arithmétiques
2. Opérateurs d'affectation
3. Opérateurs de comparaison
4. Opérateurs logiques
5. Opérateurs bit à bit

---

## 1. Opérateurs arithmétiques

Les opérateurs arithmétiques sont utilisés pour effectuer des opérations mathématiques courantes.

| Opérateur | Nom            | Description                            | Exemple |
|-----------|----------------|----------------------------------------|---------|
| `+`       | Addition       | Additionne deux valeurs                | `x + y` |
| `-`       | Soustraction   | Soustrait une valeur d'une autre       | `x - y` |
| `*`       | Multiplication | Multiplie deux valeurs                 | `x * y` |
| `/`       | Division       | Divise une valeur par une autre        | `x / y` |
| `%`       | Modulo         | Retourne le reste d'une division       | `x % y` |
| `++`      | Incrément      | Augmente la valeur d'une variable de 1 | `x++`   |
| `--`      | Décrément      | Diminue la valeur d'une variable de 1  | `x--`   |

> **Note :** En Go, `++` et `--` ne peuvent être utilisés que comme instructions, pas dans des expressions.

---

## 2. Opérateurs d'affectation

Les opérateurs d'affectation sont utilisés pour assigner des valeurs aux variables.

### Opérateur d'affectation simple :

```go
package main
import ("fmt")

func main() {
    var x = 10  // Affecte la valeur 10 à x
    fmt.Println(x)
}
```

### Opérateur d'affectation avec addition :

```go
package main
import ("fmt")

func main() {
    var x = 10
    x += 5  // Équivalent à: x = x + 5
    fmt.Println(x) // Affiche: 15
}
```

### Liste complète des opérateurs d'affectation

| Opérateur | Exemple   | Équivalent   | Description                        |
|-----------|-----------|--------------|------------------------------------|
| `=`       | `x = 5`   | `x = 5`      | Affectation simple                 |
| `+=`      | `x += 3`  | `x = x + 3`  | Affectation avec addition          |
| `-=`      | `x -= 3`  | `x = x - 3`  | Affectation avec soustraction      |
| `*=`      | `x *= 3`  | `x = x * 3`  | Affectation avec multiplication    |
| `/=`      | `x /= 3`  | `x = x / 3`  | Affectation avec division          |
| `%=`      | `x %= 3`  | `x = x % 3`  | Affectation avec modulo            |
| `&=`      | `x &= 3`  | `x = x & 3`  | Affectation avec ET bit à bit      |
| `\|=`     | `x \|= 3` | `x = x \| 3` | Affectation avec OU bit à bit      |
| `^=`      | `x ^= 3`  | `x = x ^ 3`  | Affectation avec XOR bit à bit     |
| `>>=`     | `x >>= 3` | `x = x >> 3` | Affectation avec décalage à droite |
| `<<=`     | `x <<= 3` | `x = x << 3` | Affectation avec décalage à gauche |

## 3. Opérateurs de comparaison

Les opérateurs de comparaison sont utilisés pour comparer deux valeurs.

> **Note :** La valeur de retour d'une comparaison est soit `true` (vrai) soit `false` (faux).

### Exemple :

```go
package main
import ("fmt")

func main() {
    var x = 5
    var y = 3
    fmt.Println(x > y) // Affiche: true (car 5 > 3)
}
```

### Liste des opérateurs de comparaison

| Opérateur | Nom                 | Exemple  | Description                                  |
|-----------|---------------------|----------|----------------------------------------------|
| `==`      | Égal à              | `x == y` | Retourne vrai si x est égal à y              |
| `!=`      | Différent de        | `x != y` | Retourne vrai si x est différent de y        |
| `>`       | Supérieur à         | `x > y`  | Retourne vrai si x est supérieur à y         |
| `<`       | Inférieur à         | `x < y`  | Retourne vrai si x est inférieur à y         |
| `>=`      | Supérieur ou égal à | `x >= y` | Retourne vrai si x est supérieur ou égal à y |
| `<=`      | Inférieur ou égal à | `x <= y` | Retourne vrai si x est inférieur ou égal à y |

---

## 4. Opérateurs logiques

Les opérateurs logiques sont utilisés pour déterminer la logique entre variables ou valeurs.

| Opérateur | Nom         | Description                                           | Exemple              |
|-----------|-------------|-------------------------------------------------------|----------------------|
| `&&`      | ET logique  | Retourne vrai si les deux expressions sont vraies     | `x < 5 && x < 10`    |
| `\|\|`    | OU logique  | Retourne vrai si au moins une expression est vraie    | `x < 5 \|\| x < 4`   |
| `!`       | NON logique | Inverse le résultat (vrai devient faux et vice versa) | `!(x < 5 && x < 10)` |

### Exemples :

```go
package main
import ("fmt")

func main() {
    a := 10
    b := 20
    
    // ET logique
    fmt.Println(a > 5 && b > 15) // true && true = true
    
    // OU logique
    fmt.Println(a > 15 || b > 15) // false || true = true
    
    // NON logique
    fmt.Println(!(a > 5)) // !true = false
}
```

## 5. Opérateurs bit à bit

Les opérateurs bit à bit sont utilisés sur des nombres binaires.

| Opérateur | Nom               | Description                                              | Exemple  |
|-----------|-------------------|----------------------------------------------------------|----------|
| `&`       | ET bit à bit      | Met chaque bit à 1 si les deux bits sont à 1             | `x & y`  |
| `\|`      | OU bit à bit      | Met chaque bit à 1 si au moins un bit est à 1            | `x \| y` |
| `^`       | XOR bit à bit     | Met chaque bit à 1 si seulement un des deux bits est à 1 | `x ^ y`  |
| `<<`      | Décalage à gauche | Décale les bits vers la gauche                           | `x << 2` |
| `>>`      | Décalage à droite | Décale les bits vers la droite                           | `x >> 2` |

### Exemple avec opérateurs bit à bit :

```go
package main
import ("fmt")

func main() {
    a := 5  // 0101 en binaire
    b := 3  // 0011 en binaire
    
    fmt.Println(a & b)  // 0101 & 0011 = 0001 = 1
    fmt.Println(a | b)  // 0101 | 0011 = 0111 = 7
    fmt.Println(a ^ b)  // 0101 ^ 0011 = 0110 = 6
    fmt.Println(a << 1) // 0101 << 1 = 1010 = 10
    fmt.Println(a >> 1) // 0101 >> 1 = 0010 = 2
}
```

## Priorité des opérateurs

Lorsqu'une expression contient plusieurs opérateurs, Go les évalue dans un ordre spécifique (du plus prioritaire au moins prioritaire) :

1. `()` - Parenthèses
2. `++` `--` - Incrément/Décrément (postfix)
3. `+` `-` `!` `^` `*` `&` - Unaires
4. `*` `/` `%` `<<` `>>` `&` `&^` - Multiplicatifs
5. `+` `-` `|` `^` - Additifs
6. `==` `!=` `<` `<=` `>` `>=` - Comparaison
7. `&&` - ET logique
8. `||` - OU logique

### Exemple de priorité :

```go
package main
import ("fmt")

func main() {
    result := 5 + 3*2 // Multiplication avant addition
    fmt.Println(result) // 11 (pas 16)
    
    result2 := (5 + 3) * 2 // Parenthèses changent la priorité
    fmt.Println(result2) // 16
}
```
