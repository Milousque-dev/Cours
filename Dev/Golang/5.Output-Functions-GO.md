# Go – Output Functions (Affichage)

Ce chapitre poursuit le cours en se basant sur la section **Go Output Functions** de la documentation **W3Schools**. Il détaille les fonctions d’affichage fournies par le package standard `fmt`.

---

## 1. Les fonctions d’affichage en Go

Go propose trois fonctions principales pour afficher du texte :

- `fmt.Print()`
- `fmt.Println()`
- `fmt.Printf()`

Elles appartiennent toutes au package `fmt` et sont utilisées pour afficher des données dans la sortie standard (console).

---

## 2. La fonction `fmt.Print()`

### Principe

`fmt.Print()` affiche ses arguments **sans ajouter automatiquement d’espace ni de retour à la ligne**.

### Exemple simple

```go
package main
import "fmt"

func main() {
    var i, j string = "Hello", "World"

    fmt.Print(i)
    fmt.Print(j)
}
```

Résultat :

```
HelloWorld
```

---

### Ajouter des retours à la ligne avec `\n`

```go
fmt.Print(i, "\n")
fmt.Print(j, "\n")
```

Résultat :

```
Hello
World
```

`\n` est un caractère d’échappement qui force un retour à la ligne.

---

### Afficher plusieurs valeurs avec un seul `Print()`

```go
fmt.Print(i, "\n", j)
```

Résultat :

```
Hello
World
```

---

### Ajouter un espace entre des chaînes

```go
fmt.Print(i, " ", j)
```

Résultat :

```
Hello World
```

---

### Comportement avec des valeurs non textuelles

Lorsque les arguments **ne sont pas des chaînes**, `Print()` ajoute automatiquement un espace.

```go
var a, b = 10, 20
fmt.Print(a, b)
```

Résultat :

```
10 20
```

---

## 3. La fonction `fmt.Println()`

### Principe

`fmt.Println()` fonctionne comme `Print()` mais :

- ajoute automatiquement un **espace** entre les arguments
- ajoute automatiquement un **retour à la ligne** à la fin

---

### Exemple

```go
package main
import "fmt"

func main() {
    var i, j string = "Hello", "World"

    fmt.Println(i, j)
}
```

Résultat :

```
Hello World
```

`Println()` est la fonction la plus utilisée pour un affichage simple et lisible.

---

## 4. La fonction `fmt.Printf()`

### Principe

`fmt.Printf()` permet d’**afficher du texte formaté** à l’aide de *verbes de formatage*.

Elle ne génère **pas automatiquement** de retour à la ligne (sauf si `\n` est précisé).

---

### Verbes de base

- `%v` : valeur par défaut
- `%T` : type de la valeur

### Exemple

```go
package main
import "fmt"

func main() {
    var i string = "Hello"
    var j int = 15

    fmt.Printf("i a pour valeur : %v et type : %T\n", i, i)
    fmt.Printf("j a pour valeur : %v et type : %T", j, j)
}
```

Résultat :

```
i a pour valeur : Hello et type : string
j a pour valeur : 15 et type : int
```

# 5. Verbes de formatage (Formatting Verbs)

## 5.1 Verbes généraux (tous types)

| Verbe | Description            |
|-------|------------------------|
| `%v`  | Valeur par défaut      |
| `%#v` | Valeur en syntaxe Go   |
| `%T`  | Type de la valeur      |
| `%%`  | Affiche le symbole `%` |

### Exemple

```go
var i = 15.5
var txt = "Hello World!"

fmt.Printf("%v\n", i)
fmt.Printf("%#v\n", i)
fmt.Printf("%v%%\n", i)
fmt.Printf("%T\n", i)

fmt.Printf("%v\n", txt)
fmt.Printf("%#v\n", txt)
fmt.Printf("%T\n", txt)
```

---

## 5.2 Verbes pour les entiers

| Verbe  | Description                        |
|--------|------------------------------------|
| `%b`   | Base 2                             |
| `%d`   | Base 10                            |
| `%+d`  | Base 10 avec signe                 |            
| `%o`   | Base 8                             |
| `%O`   | Base 8 avec `0o`                   |
| `%x`   | Base 16 (minuscule)                |
| `%X`   | Base 16 (majuscule)                |
| `%#x`  | Base 16 avec `0x`                  |
| `%4d`  | Largeur 4, aligné à droite         |
| `%-4d` | Largeur 4, aligné à gauche         |
| `%04d` | Largeur 4, complété avec des zéros |

---

## 5.3 Verbes pour les chaînes (`string`)

| Verbe  | Description                 |
|--------|-----------------------------|
| `%s`   | Chaîne simple               |
| `%q`   | Chaîne entre guillemets     |
| `%8s`  | Largeur 8, alignée à droite |
| `%-8s` | Largeur 8, alignée à gauche |
| `%x`   | Hexadécimal                 |
| `% x`  | Hexadécimal avec espaces    |

---

## 5.4 Verbe pour les booléens

| Verbe | Description  |
|-------|--------------|
| `%t`  | true / false |

```go
fmt.Printf("%t", true)
```

---

## 5.5 Verbes pour les flottants

| Verbe   | Description            |
|---------|------------------------|
| `%e`    | Notation scientifique  |
| `%f`    | Décimal                |
| `%.2f`  | Précision 2            |
| `%6.2f` | Largeur 6, précision 2 |
| `%g`    | Notation compacte      |
